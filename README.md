# Vehículos en EUA

# Descripción
Aplicación web desarrollada con Streamlit para visualizar y analizar datos de anuncios de venta de vehículos en Estados Unidos.
Permite explorar información como kilometraje, año del modelo y precio mediante gráficos interactivos.

# Funcionalidades
Histograma del kilometraje de los vehículos
Gráfica de dispersión entre año del modelo y precio
Interfaz interactiva con selección de gráficos mediante checkboxes

# Tecnologías utilizadas
Python
Streamlit
Pandas
Plotly Express
Cómo ejecutar el proyecto
streamlit run app.py
